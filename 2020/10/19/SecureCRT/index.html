设置

为了防止世界被破坏 为了维护世界的和平 我需要设置一下 SecureCRT 🤣

关闭标签的标题变更功能

设置后可以使打开的标签的标题显示成 session 的名称，方便辨识服务器

  1. 点击菜单栏 Options
  2. 点击 Edit Default Session…
  3. Terminal > Emulation > Advanced
  4. 选中 Ignore window title change requests
  5. 点击 OK 选择 Change ALL Sessions
